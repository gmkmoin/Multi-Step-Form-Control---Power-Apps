# 📝 Travel Request Management App

A modern **Multi-Step Travel Request Management Application** built using **Microsoft Power Apps** and **SharePoint**, designed to simplify the process of creating, reviewing, and managing employee travel requests.

The application provides an intuitive step-by-step wizard for users to submit travel requests while maintaining data consistency through validations and centralized SharePoint storage.

---

# 🚀 Features

## 📋 Dashboard

- Displays all submitted travel requests
- Clean tabular layout for easy navigation
- Search travel requests
- Filter travel requests
- Sort request records
- View request details
- Create new travel request
- Edit existing travel request
- Centralized travel request management

---

## 🧾 Multi-Step Travel Request Form

The application follows a guided wizard approach where users complete information section-by-section.

### Step 1 – Employee Information 👤

- Employee Name
- Employee Email
- Employee ID
- Department
- Business Unit
- Basic requester information
- Required field validations

---

### Step 2 – Travel Details ✈️

- Travel From
- Travel To
- Destination
- Travel Purpose
- Departure Date
- Return Date
- Business justification
- Date validation

---

### Step 3 – Hotel Details 🏨

- Hotel Name
- Hotel Address
- Check-In Date
- Check-Out Date
- Hotel requirements
- Accommodation information

---

### Step 4 – Review & Submit ✅

- Review complete request
- Verify entered information
- Edit previous steps if required
- Final submission
- Save request into SharePoint List
